# Math4AI documentation

Public documentation for **Math4AI** — a Python plotting toolkit for mathematics and machine learning figures.

- Documentation site: https://math4ai-plot.readthedocs.io/
- Package: `pip install math4ai`

This repository contains **Markdown docs only**. It should not include runtime Python source files.

## Edit workflow

Docs are authored from the `math4ai-plot` source tree, then synced to this public docs repo.

Typical local flow:

```bash
cd ~/math4ai-plot
# edit README/dev.md/source docs notes
# then update pages in ~/math4ai-plot-docs/docs
```

If pages and API differ, prefer source-of-truth from `math4ai-plot`.
