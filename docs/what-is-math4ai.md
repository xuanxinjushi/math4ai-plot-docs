# What is Math4AI?

Math4AI is a focused plotting library that sits between raw matplotlib code and full charting frameworks.

Its purpose is to make chapter figures look coherent across an entire technical manuscript.

## Core idea

In math-heavy projects, each figure script is often written at a different time with slightly different style choices. Math4AI reduces style drift by centralizing:

- Colors
- Matrix and vector drawing conventions
- Mindmap iconography
- Figure save behavior

## Typical users

- Authors of mathematics and machine learning books
- Educators producing polished lecture diagrams
- Researchers creating consistent figure sets for papers or reports

## Scope

Math4AI is not a general dashboarding or plotting framework. It is a lightweight utility layer on top of matplotlib and numpy for script-based figure pipelines.
